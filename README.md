# Ocean Fantasy

A **2D Game** where the diver collects **Marine Litter** and prevents kinds of fish eating them.

## Environment

- Unity 2018.3.9f1

## Control (*PC*)

- Use **WASD / Arrow Keys** to control the diver **move**.
- Use **Space Key** to **accelerate**.
